## About

W ramach projektu ujęto wizualizacje następujących algorytmów sortowania:

| Algorytm       | Złożoność             |
| -------------- | --------------------- |
| Selection sort | $$\theta{(n^2)}$$     |
| Bubble sort    | $$\theta{(n^2)}$$     |
| Insertion sort | $$\theta{(n^2)}$$     |
| Quick sort     | $$\theta{(nlog(n))}$$ |
| Counting sort  | $$\theta{(n+k)}$$     |
